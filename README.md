## 💡 Sobre o projeto

Este projeto faz parte da minha jornada no curso de **Front-End da Comunidade Dev Completo**, que foi o meu primeiro contato com o desenvolvimento front-end. O foco aqui foi entender na prática como o JavaScript se comunica com o HTML através de eventos e manipulação do DOM — conceitos que formam a base de qualquer interface interativa.

Durante o estudo do curso, quando a professora sugeriu o desafio de criar um site to-do list, a primeira coisa que pensei foi em relacionar com os livros, já que eu amo ler.

A proposta do exercício era praticar JavaScript interativo com formulários — e em vez de fazer mais uma lista de tarefas genérica, transformei em uma estante virtual inspirada no Kindle.

O resultado é uma aplicação onde é possível adicionar livros, marcar como lidos e avaliar cada um deles. Simples, funcional e com a minha cara. 🧡
